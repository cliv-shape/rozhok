### Бот, который умеет делать ДЕМОТИВАТОРЫ и писать в чате!

# Установка & запуск:
```
npm i
cd src 
pm2 start index.js
# Закрыть процессы - pm2 stop all
# Перезагрузить процессы - pm2 reload all
```
