# rozhok
Рожок имеет две версии
- v1 - запоминание текста и воспроизведение его, создание демотиваторов и сохранение изображений
- v2 - чат-бот с использованием API xu.su

В каждой папке есть инструкция по запуску, но сначало нужно установить модули собственна..
```npm i```
, дальше следуйте инструкции по запуску и не забывайте про конфиг..
